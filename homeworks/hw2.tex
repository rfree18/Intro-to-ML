\paragraph{1.} After replacing the label set from $\left\{ 0,1 \right\}$ to
$\left\{ -1,1 \right\}$, we introduced the log loss
\begin{align*}
    D_{\log}(y, \vx; M) = \frac{1}{\log 2} \log(1+\exp(-s(y, \vx; M))),
\end{align*}
as an alternative to the logistic regression distance function above. Show that
these two are equivalent up to a constant multiplication for logistic
regression.

\vspace{5mm}

\paragraph{2.} Unlike the log loss, the hinge loss, defined below, is not
differentiable everywhere:
\begin{align*}
    D_{\hinge}(y, \vx; M) = \max(0, 1 - s(y, \vx; M)).
\end{align*}
Does it mean that we cannot use a gradient-based optimization algorithm for
finding a solution that minimizes the hinge loss? If not, what can we do about
it?
\\
\\
SOLUTION: For points $ > 1$, the derivative is 0. For points $< 1$, the hinge function is equal to $1 - y\vw^\top \tilde{\vx}$. This case has a trivial derivative of $-y \tilde{\vx}$. The only part which we cannot find the derivative is at the point where $s(y, \vx; M) = 1$. Instead, we can simply arbitrarily define the derivative to be 0 at that point because as far as we are concerned, a value should be considered correctly classified if it is equal to exactly 1. Anything greater is a correct classification and values slightly below are barely classified. 
\vspace{5mm}

    \gamma =& \frac{1}{2} (d^+ + d^-).
\end{align*}
Show that minimizing the norm of the weight vector of a support vector machine
